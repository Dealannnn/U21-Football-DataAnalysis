## Contexte du Projet
Ce projet a été réalisé dans le cadre du module **"Outils libres pour le développement logiciel"** (Master 1 Big Data - IED Paris 8). 

L'objectif métier est de développer un algorithme d'analyse de données axé sur le **Scouting (recrutement sportif)**. À partir d'un dataset volumineux issu de Kaggle, l'outil filtre, analyse et compare les statistiques (techniques, physiques et financières) des joueurs de football professionnels pour identifier des "pépites" : des joueurs de moins de 21 ans (U21) présentant des profils statistiques similaires aux joueurs de très haut niveau, tout en respectant des critères de valeur marchande.

## Stack Technique
Ce projet s'appuie exclusivement sur des outils et librairies open source :
* **Langage :** Python
* **Environnement :** Jupyter Notebook
* **Manipulation des données :** Pandas, NumPy
* **Data Visualisation :** Matplotlib, Seaborn
* **Gestion de version :** Git / GitHub

## Fonctionnalités Principales
1. **Data Cleaning & Préparation :** Nettoyage des données brutes, gestion des valeurs manquantes et formatage des types de données.
2. **Filtrage Ciblé (Scouting) :** Isolement de la cohorte U21 et application de filtres financiers (Valeur marchande, salaire).
3. **Analyse Exploratoire (EDA) :** Étude des corrélations entre les différentes caractéristiques des joueurs.
4. **Data Visualisation :** Génération de graphiques avancés (radars, scatter plots, bar charts) pour comparer les profils des jeunes talents.
